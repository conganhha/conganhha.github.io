## Hello, World!

Here's how you might write a simple "Hello, World!" in Python:

```python
def greet(name="World"):
    return f"Hello, {name}!"

print(greet())  # Outputs: Hello, World!
```

``cpp
cout << "Hello, World!" << endl; // Outputs: Hello, World!
```
